## Как настроить eSIM на iPhone

Настройка eSIM на iPhone едина для всех моделей от XR до 17 Pro Max. Различия минимальны и касаются только версии iOS.

### Способ 1: сканировать QR-код (основной, самый быстрый)

1. Купите eSIM у провайдера (Airalo, Holafly и другие). Получите QR-код на email.
2. Откройте письмо на компьютере или втором телефоне.
3. На iPhone: Настройки → Сотовая связь → Добавить eSIM.
4. Нажмите «Сканировать QR-код».
5. Наведите камеру на QR-код.
6. Дождитесь загрузки профиля (30 секунд – 2 минуты).
7. Назначьте метку eSIM: «Путешествия», «Работа», «Основной».
8. Выберите только что добавленную eSIM для сотовых данных.
9. Включите «Роуминг данных» для eSIM — это техническая особенность, реальный роуминг не используется.

### Способ 2: ввести данные вручную

Если QR-код не сканируется (повреждён, слишком мелкий, нет второго устройства для отображения):
1. Настройки → Сотовая связь → Добавить eSIM.
2. Нажмите «Ввести данные вручную» внизу экрана.
3. Введите SM-DP+ адрес и код активации из письма провайдера.
4. Нажмите «Далее» и дождитесь активации.

### Способ 3: через приложение провайдера

Самый простой способ для новичков:
1. Установите приложение провайдера (Airalo, Holafly и другие) из App Store.
2. Купите тариф в приложении.
3. Нажмите «Установить eSIM» — приложение само направит вас в настройки и установит профиль.

Подробная инструкция с фото всех шагов — [Как настроить eSIM на iPhone и Android](/blog/how-to-setup-esim/).

## Dual eSIM на iPhone: два номера, один телефон

Начиная с iPhone 15 (и iPhone 14 в США), вы можете использовать две активные eSIM одновременно — полностью отказавшись от физической SIM. Это означает:

- Российский номер через eSIM МТС/Билайн/МегаФон
- Туристический eSIM для интернета через Airalo или другие
- Оба номера принимают звонки и SMS одновременно

Настройка:
1. Добавьте обе eSIM через Настройки → Сотовая связь → Добавить eSIM.
2. Назначьте метки: «Основной» (российский номер) и «Путешествия» (туристический eSIM).
3. Выберите основную SIM для звонков и отдельно — для сотовых данных.
4. Включите «Переключение сотовых данных» — телефон будет автоматически использовать ту eSIM, где лучше сигнал для интернета.

### Сценарий: путешествия + домашний номер

Вы летите в Турцию. Российская eSIM (МТС) остаётся активной для входящих звонков от семьи и SMS от банка. Туристическая eSIM (Roami Турция, активирована с web20) обеспечивает интернет. Вы не платите за роуминг МТС, потому что для данных используется местный eSIM.

### Сценарий: работа + личный

Два номера на одном iPhone: рабочий (eSIM Билайн) и личный (eSIM МегаФон). Оба принимают звонки. Для интернета используете тот, где лучше покрытие в данный момент.

## iPhone eSIM: частые проблемы и решения

### Ошибка активации «Не удалось активировать eSIM»

Причины:
- Неверный EID. Проверьте EID в Настройки → Основные → Об устройстве. Сверьте с провайдером — профиль может быть привязан к неверному EID.
- Просроченный QR-код. Некоторые провайдеры ограничивают срок действия QR-кода 24 часами. Запросите новый код.
- Отсутствие интернета. Для загрузки профиля eSIM нужен Wi-Fi. Переключитесь на стабильную сеть.

### Нет интернета после активации

1. Проверьте, что eSIM выбрана для сотовых данных: Настройки → Сотовая связь → Сотовые данные → выберите eSIM.
2. Включите «Роуминг данных» для eSIM.
3. Проверьте APN: Настройки → Сотовая связь → eSIM → Сотовая сеть → APN. Правильные настройки APN должны быть в письме от провайдера.
4. Перезагрузите телефон.
5. Подождите 5–10 минут после активации — некоторым eSIM нужно время для регистрации в сети.

### eSIM пропала после обновления iOS

Редкая, но известная проблема. Решение:
1. Перезагрузите телефон.
2. Если eSIM не появилась — удалите профиль eSIM в Настройки → Сотовая связь.
3. Установите eSIM заново через приложение провайдера или QR-код.

После крупных обновлений iOS (например, с 18 на 19) Apple рекомендует переустановить eSIM, если она не отображается.
