Ошибки при использовании eSIM в Филиппинах

Большинство проблем с eSIM на Филиппинах связано с островной спецификой и настройками. Вот что чаще всего идёт не так.

1
Для работы eSIM нужно включить «Роуминг данных». Туристические eSIM технически работают через роуминговых партнёров — без включения опции «Роуминг данных» интернет не появится, даже если eSIM активирована. Это не настоящий роуминг, деньги с домашнего счёта не списываются.
2
На телефоне с двумя SIM нужно выбрать eSIM для интернета. После установки eSIM телефон может продолжить использовать домашнюю SIM для передачи данных — и вы будете платить за роуминг. В настройках нужно явно выбрать eSIM как линию для сотовых данных и отключить автоматическое переключение между SIM.
3
eSIM нужно установить до вылета, на домашнем Wi-Fi. Для активации eSIM требуется стабильное интернет-соединение для загрузки профиля. Wi-Fi в аэропортах часто перегружен, ограничен по времени или требует SMS-подтверждения — установить eSIM по прилёту может не получиться.
4
